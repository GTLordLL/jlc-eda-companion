```
        → 测试 TP_3V3 是否为 3.3V±5%
Step 2: 焊接 MCU 最小系统 → 烧录 Blink 程序
...
```

### 2.3 常见故障排查决策树

```
现象：MCU 无法烧录程序
├── 检查供电 → VCC 是否为 3.3V？
│   ├── 否 → 检查 LDO 输入/输出电容是否焊反
│   └── 是 → 下一步
├── 检查复位脚 → NRST 是否为高电平？
│   ├── 否 → 上拉电阻缺失/焊反
│   └── 是 → 下一步
└── 检查烧录接口 → SWDIO/SWCLK 是否连通？
```
